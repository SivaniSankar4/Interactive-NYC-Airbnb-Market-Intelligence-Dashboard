This project, titled “Interactive NYC Airbnb Market Intelligence Dashboard,” focuses on analysing Airbnb listing data from New York City using Microsoft Power BI, a powerful data visualization and analytics tool. The dataset contains detailed information about Airbnb properties, including neighbourhood group, neighbourhood, room type, price, availability throughout the year, and customer reviews. By leveraging Power BI’s data modelling, DAX calculations, and interactive visualization capabilities, the project transforms raw data into meaningful insights.
The primary aim of this project is to design and develop an interactive Power BI dashboard that provides a comprehensive overview of the NYC Airbnb market. The dashboard presents key performance indicators (KPIs) such as total listings, total reviews, average price, average availability, most expensive neighbourhood, and most popular room type. In addition, the dashboard includes comparative charts, geographical maps, scatter plots, and donut charts to analyse pricing trends, demand patterns, geographic distribution, and customer preferences.
Furthermore, interactive slicers and navigation features are incorporated to allow users to dynamically filter and explore the data based on neighbourhood groups, room types, price categories, and availability categories. This enhances user experience and enables deeper, customized analysis. The project not only demonstrates technical skills in Power BI and DAX but also highlights the importance of data-driven decision-making in the modern hospitality and tourism industry.
Overall, this project serves as a practical application of data analytics concepts and showcases how business intelligence tools can be effectively used to uncover insights, identify trends, and support strategic planning in real-world scenarios such as the Airbnb marketplace
